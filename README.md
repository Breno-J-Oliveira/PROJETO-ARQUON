# SPRINT 1

A Sprint 1 corresponde à fase inicial de planejamento do projeto Arqon.
Nesta etapa foram definidos os requisitos do sistema, identidade visual e estrutura inicial da plataforma.

O sistema tem como objetivo desenvolver uma plataforma digital para locação de roupas premium sustentáveis, permitindo que usuários acessem peças de marcas renomadas de forma prática e moderna.

O projeto contempla cinco páginas principais:

* Página de Login
* Cadastro de Marcas Parceiras
* Cadastro de Peças
* Painel Administrativo
* Listagem de Produtos

O objetivo é desenvolver um sistema acessível, organizado e eficiente para gerenciamento de peças de vestuário e marcas parceiras.

---

## Funcionalidades

* Login seguro com e-mail e senha
* Cadastro de marcas parceiras
* Cadastro de produtos (peças de roupa)
* Registro de informações detalhadas das peças
* Listagem organizada dos produtos cadastrados
